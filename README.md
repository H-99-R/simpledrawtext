# simpledrawtext
Simple Draw Text Header-Only Class for Basic OpenGL Applications

# Requirements
- FreeGlut
- CMake

# Supported OS & Tested Compilers
- Windows 10 [VS 2019]
- Ubuntu 19.10 [gcc (Ubuntu 9.2.1-9ubuntu2) 9.2.1]
